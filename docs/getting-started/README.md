# Getting Started with azure-analyzer

Welcome! This section covers installation, first run, and common scenarios.

## New to azure-analyzer?

Start with these three steps:

1. **[Installation](installation.md)** — Clone the repo and import the module.
2. **[First Run](first-run.md)** — Run your first assessment in 2 minutes.
3. **[Common Scenarios](quickstart-scenarios.md)** — Pick your use case (subscription scan, repo scan, customize).

## What You'll Get

After your first run:
- `report.html` — Interactive findings browser with sorting, filtering, and heatmaps.
- `report.md` — Markdown findings for Git workflows.
- `output/results.json` — Structured findings in Schema 2.2.
- `output/entities.json` — Deduplicated cross-tool entities.
